### TASK
Let's make a modern SWIFT based version of this app for MacOS 26 with GlassUI and OS native features that we can make use of. This is app is originally built for Windows, and it's ported to MacOS, but it's quite dated and have few issues causing the app to be unstable with modern Macs, especially Apple Silicon based ones.

### CONTEXT
Original App website: https://ipmsg.org/
MacOS port website: https://ishwt.net/en/software/ipmsg/

Apple Developer Documentation: https://developer.apple.com/documentation

### CONSTRAINTS
We want the app to fully natively built using MacOS tools and native OS APIs and features.


---


The app seems to be running fine, I can see the online user. However, I don't see following features yet.

1) We need toast notification when the message comes in.
2) I don't see any mechanism to send message. Since there is no sending mechanism, there's no file transfer mechanism either. File transfers should also be visible in the messaging with message history, and if any images are transferred, there should be an image viewer.
3) There is no history being shown for messages sent by another user from previous session.
4) The icon dropdown shows the number of users online, and unread messages number even after reading the messages. It doesn't show any way to invoke the messaging window.
5) The messaging window is visible in task switcher (CMD + TAB). Ideal behaviour should be app icon in task switcher when the window is open, but not visible if the window is closed.


---

Message sending works... but,

1) The message history only shows incoming messages, not outgoing. It should ideally show a regular chat window like layout with distinct incoming and outgoing message bubbles.
2) The menu bar dropdown should have cleaner layout.
   - Top section, online status and absence mode dropdown
   - section 2, should be menu item with icon (not button), so it looks like native menu. It should be "New Message", "Main Window" and "Message History".
   - We can remove the recent message section from dropdown.
3) There are few icons in the WORKSPACE/icons folder, please use whichever fitting icons you can use in the interface. Also there's app icon that can be used for task switcher and menu bar icons.
